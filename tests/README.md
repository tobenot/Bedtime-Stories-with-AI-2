# 测试说明

# 分支命名逻辑测试

# 运行测试

1. `node tests/branch-naming.test.mjs`

# 需求说明

1. 没有"（分支X）"后缀时，生成"基础标题 + （分支）"。
2. 有"（分支）"后缀时，生成"（分支2）"。
3. 有"（分支X）"后缀时，生成"（分支X+1）"。
4. 长度计算按显示长度处理：半角字符算0.5，全角字符算1。
5. 最终标题显示长度不超过 `MAX_TITLE_LENGTH`（26）。
6. 超过限制时先截断基础部分，再拼接后缀。

# 测试场景

1. 基础标题生成分支
2. 从"（分支）"生成"（分支2）"
3. 从"（分支2）"生成"（分支3）"
4. 大数字分支（如"（分支99）"）
5. 长标题截断
6. 连续从原始标题分叉
7. 从分支标题继续分叉
8. 边界情况：44个半角字符（显示长度22）
9. 边界情况：52个半角字符（显示长度26）
